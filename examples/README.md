# CLAUDE.md 記入例集（士業向け）

`CLAUDE.md.template` を士業ごとにどう埋めるかの記入例です。

## 利用方法

自所に近い例を `CLAUDE.md` としてコピーし、自所情報に書き換えてください。

```bash
# 例：行政書士の場合
cp examples/gyosei-shoshi/CLAUDE.md ./CLAUDE.md
```

## 士業別の記入例

| ディレクトリ | 想定士業 | カバーする主要観点 |
|---|---|---|
| [`gyosei-shoshi/`](./gyosei-shoshi/CLAUDE.md) | 行政書士 | 建設業許可・産廃許可・相続中心、業際・守秘義務 |
| [`zeirishi/`](./zeirishi/CLAUDE.md) | 税理士 | 法人顧問・確定申告・相続税、税理士独占業務 |
| [`sharoshi/`](./sharoshi/CLAUDE.md) | 社会保険労務士 | 労務顧問・給与計算・助成金、労働紛争範囲 |

## 自分の士業区分が見当たらない場合

最も近い例をベースに、士業特有のセクションを追加してカスタマイズしてください。例えば：

- **司法書士**：登記分野の取扱（不動産登記 vs 商業登記）、簡裁代理権、裁判所提出書類
- **弁理士**：特許・商標・意匠の取扱比率、外国出願対応、技術分野の専門性
- **中小企業診断士**：コンサルティング領域、補助金支援、行政書士との兼業
- **公認会計士**：監査業務 vs アドバイザリー、規模別の取扱、税理士登録の有無

「この士業の例も追加してほしい」という要望は、Issues か Pull Request で歓迎します。
